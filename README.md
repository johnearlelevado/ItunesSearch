# ItunesSearch
  - I have hardcoded the movie as entity for simplicity
  - The architecture is MVVM, I chose this over MVP/MVC because it helps reinforce one of SOLID principles which is Single responsiblity principle.

### Tech stack
  - Retrofit2
  - Dagger Hilt
  - View Binding
  - MVVM Pattern
  - ViewModel
  - LiveData
  - RxJava
  - Room DB
